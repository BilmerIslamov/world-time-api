## Installation
```
python3 -m pip install --user virtualenv
# or
https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/
```
## Install libraries
```
pip install asgiref
pip install django
pip install djangorestframework
pip install pillow
pip install pytz
pip install sqlparse
pip install tzdata
```
